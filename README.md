# EXLI Practices Prototype

## Setting up with PhoneGap:

- `phonegap create exli-practices`
- `cd exli-practices`
- `rm -rf www`
- `git clone git@bitbucket.org:colabcoop-ondemand/exli-prototype.git www`

## Developing:

All development is done from the `/app` folder.

From project root:

- `cd app`
- `npm install`
- `gulp`

Hack away.
